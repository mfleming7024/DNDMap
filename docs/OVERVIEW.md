# Project Overview

The D&D Map project is a full-stack JavaScript application that allows users to view and interact with maps from a Dungeons & Dragons campaign.

## Features

*   Interactive map powered by Leaflet.js.
*   Custom markers for points of interest.
*   Ability to add and remove markers.
*   Data is stored in a MongoDB database.
*   A generative AI feature to answer questions about the world.
