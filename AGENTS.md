# D&D Map Project Agent

This document provides a guide for understanding and working with the D&D Map project.

## Overview

This project is a web application for visualizing Dungeons & Dragons game worlds with interactive maps. For a detailed overview, see [docs/OVERVIEW.md](./docs/OVERVIEW.md).

## Getting Started

To get the project running on your local machine, follow the instructions in [docs/GETTING_STARTED.md](./docs/GETTING_STARTED.md).

## Architecture

The project is a single-page application with a Node.js backend. Learn more about the architecture in [docs/ARCHITECTURE.md](./docs/ARCHITECTURE.md).
